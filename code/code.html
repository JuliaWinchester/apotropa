Listed below are software projects I’ve created or contributed to, as well as published downloadable datasets of 3D anatomical surface meshes that I’m associated with. Also check my Github for more coding projects.

Code

MorphoTester: Desktop app for quantifying topographic shape from 3D meshes representing biological surfaces, most commonly (so far) teeth but also other elements. Built in Python using Numpy and Scipy for numerical computing, PyQt4 for UI design, and Mayavi for 3D visualization. Publication link. Published analyses using this software: 1, 2, 3, 4, 5, 6, 7.

auto3dgm: R and Matlab packages to automatically align 3D surfaces and generate corresponding surface psuedolandmarks. Minimum spanning trees of shape are also derived, which makes it possible to trace a continuum of shape variation from one extreme shape specimen to another through a series of intermediate shapes linking the two extremes. The Matlab package is cluster capable. Publication link. Published analyses using this software: 1, 2, 3, 4, 5, 6.

hecate: Matlab/Python package to consistently segment of biological surface regions. This software attempts to use an algorithm to objectively get at something biologists are commonly interested in: the identification of discrete regions or features on a surface that are shaped similarly across a collection of surfaces. WIP.

Xfix: An example single-page web application for anatomical education, where students can access images of anatomical specimens with dynamically-drawn labels pointing to specific structures. Features include relevant image suggestion, interactive quiz mode, and a full on-the-fly editor mode for instructors. Built with a MySQL database, a PHP back-end, and a front-end using AngularJS, Angular Material for UI, and PaperJS for displaying and editing images and labels. This site requires login due to bandwidth concerns, but I’m happy to provide login info on request.

Data

I’m currently part of the team working on MorphoSource, an online repository where users can upload, download, and share 3D digital data. In addition, I’m associated with a number of downloadable datasets of 3D scans of second mandibular molar teeth stored on MorphoSource, listed below.

Winchester et al. 2014: 111 platyrrhine primate specimens. Associated paper.

Bunn et al. 2011: 16 strepsirrhine primate specimens. Associated paper.

Pampush et al. molaR: 15 platyrrhine primate specimens, partially overlaps with the Winchester et al. 2014 sample. Associated paper.
